+.tmap {tmap}R Documentation

Stacking of tmap elements

Description

The plus operator allows you to stack tmap-elements, and groups of tmap-elements.

Usage

## S3 method for class 'tmap'
e1 + e2

Arguments

e1

first tmap-element

e2

second tmap-element

References

Tennekes, M., 2018, tmap: Thematic Maps in R, Journal of Statistical Software, 84(6), 1-39, DOI
